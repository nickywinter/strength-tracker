## GitHub Pages
1. Create a repo
2. Upload all files to the root
3. Go to Settings -> Pages
4. Deploy from branch: main / root
5. Open the GitHub Pages URL on iPhone and Add to Home Screen

## Notes
- Data is stored in localStorage in your browser
- Use the backup box to export and restore your data
